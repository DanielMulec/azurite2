# Research Sources

## Purpose

This catalog tracks useful sources discovered while researching, designing, and
building Azurite. It is a reusable starting point, never an exhaustive or
exclusive source list.

## How To Use This Catalog

- Reuse relevant sources when they help with the current task.
- Look for additional current sources whenever research affects product,
  architecture, security, dependencies, standards, or implementation.
- Prefer primary sources such as official documentation, standards, source
  repositories, research papers, and maintainer-written material.
- Add a source only when it materially informs a decision, implementation, or
  product direction.
- Keep notes concise and capture context, caveats, and access dates.
